# 官方接口 - 前置介绍

该系列后面所有文档如无特殊说明均假设你已经成功执行以下步骤：

* `conf = WechatConf(...)`
* `wechat = WechatBasic(conf=conf)`

均默认调用 `wechat` 变量的类成员方法。
